# SD Day Traders

Starter landing page for a San Diego day trading community focused on interaction
and meetups.

## Preview locally

Run a static server from the project folder:

```sh
python -m http.server 4173
```

Then open `http://localhost:4173`.

## Customize

- Edit `index.html` to swap in your real invite links and meetup copy.
- Replace `assets/sd-day-traders-hero.jpeg` if you want to swap the pulled group
  artwork for a different image later.
- Tweak colors and layout in `styles.css`.
